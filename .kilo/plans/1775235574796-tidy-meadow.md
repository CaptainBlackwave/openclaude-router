# Plan for Smart Router Optimization: Free LLM Models Only

## Objective
Create an optimized smart_router that exclusively uses the most efficient and available free LLM models, with automatic switching and routing based on performance metrics.

## Current Codebase Analysis
- **smart_router.py**: Implements a SmartRouter class that routes requests to providers (OpenAI, Gemini, Ollama, Atomic-Chat) based on latency, cost, health.
- Providers include paid (OpenAI: $0.002/1k, Gemini: $0.0005/1k) and free (Ollama, Atomic-Chat: $0.0/1k).
- Routing strategy: latency, cost, balanced.
- Fallback on failure.
- Local providers: Ollama (http://localhost:11434), Atomic-Chat (http://127.0.0.1:1337).
- Config via .env: ROUTER_MODE, ROUTER_STRATEGY, etc.

## Research on Free LLM Models (2026)
Based on current leaderboards and API providers:
- **Top Models**:
  - GLM-5 (Z AI): #1 in February 2026 rankings, high reasoning score.
  - Kimi K2.5 (Reasoning): Previously top, strong in math and coding.
  - MiniMax-M2: New entrant, commercially usable.
  - Olmo (Allen AI): Fully open-source, excellent in reasoning, coding, math.
  - GPT-OSS-120B (OpenAI): Open-weight model for powerful reasoning.
- **API Providers**:
  - Hugging Face Inference API: Free tier with rate limits.
  - Replicate: Hosts open-source models, some free or low-cost.
  - SiliconFlow: Specializes in open-source LLM APIs.
  - LocalAI: Drop-in OpenAI alternative, runs models locally.
  - Together AI: Free tier for open-source models.
  - DeepInfra: Free inference for various models.
  - OpenRouter: Aggregator with free credits/trials for various models.
  - ApiFreeLLM: Free unlimited API with OpenAI compatibility, rate limited to 1 req/25s.
  - LLMAPI.AI: Unified free API for multiple LLMs, OpenAI compatible.
  - Other notable: Fireworks AI, Novita AI, Hypereal AI.
- **Existing free providers**: Ollama (runs local models like Llama, Qwen), Atomic-Chat (Apple Silicon optimized).
- **Efficiency metrics**: Prioritize models with low latency, high throughput, good accuracy on tasks.

## Proposed Changes
1. **Filter to Free Providers Only**: Modify build_default_providers to exclude paid providers, or add a flag to enable free-only mode.
2. **Add New Free Providers**:
   - Olmo via Hugging Face Inference API.
   - GLM-5 via SiliconFlow, Replicate, or OpenRouter API.
   - Kimi K2.5 via available API (e.g., OpenRouter).
   - LocalAI as a local provider alternative to Ollama.
   - Replicate for additional open-source models.
   - Together AI for free open-source models.
   - DeepInfra for free inference.
   - OpenRouter for aggregated free models.
   - ApiFreeLLM for free unlimited OpenAI-compatible API.
   - LLMAPI.AI for unified free access to multiple LLMs.
3. **Optimization**:
   - Benchmark models on efficiency (latency, token/s).
   - Auto-switch based on real-time performance.
   - Prioritize local models for lowest latency.
4. **Config Updates**: Add FREE_ONLY_MODE=true, list of free models and API keys.
